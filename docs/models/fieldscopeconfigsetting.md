# FieldScopeConfigSetting

The setting of the datapoint in the scope config that you configured in the Kombo dashboard.

## Example Usage

```typescript
import { FieldScopeConfigSetting } from "@kombo-api/sdk/models";

let value: FieldScopeConfigSetting = "ENABLED";
```

## Values

```typescript
"ENABLED" | "DISABLED" | "OPTIONAL"
```
